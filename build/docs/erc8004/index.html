4. Relationship to agent.json​

The ERC-8004 registration file (erc8004.json) serves as the on-chain identity pointer for SigmaGrid. It declares the canonical location of the agent descriptor file (agent.json), which provides bot-facing capability information.

  • erc8004.json = on-chain identity pointer
  • agent.json = bot-facing capability descriptor
  • ERC-8004 identity tokenURI should point to erc8004.json

5. Notes​

  • Registration file mutability: The registration file (erc8004.json) is mutable off-chain, allowing for updates as needed.
  • On-chain identity stability: The on-chain identity remains stable, ensuring consistent identification.
  • Test data: Some data within this registration is intended for testing purposes.
  • AgentQuant exposure: There is no exposure to AgentQuant.
